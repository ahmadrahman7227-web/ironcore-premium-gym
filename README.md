# IRONCORE — Premium Gym Landing Page

IRONCORE is a modern cinematic gym landing page built with React, Vite, Tailwind CSS, and Framer Motion.  
This project focuses on creating a luxury fitness brand experience through premium UI design, smooth animations, responsive layouts, and immersive interactions.

---

# Preview

IRONCORE is designed to feel like a high-end fitness campaign website with:
- cinematic dark aesthetic
- luxury typography
- immersive sections
- smooth animations
- modern frontend interactions
- premium mobile experience
